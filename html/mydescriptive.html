mydescriptive

Description

It gives the general summary statistics: no. of observations, average, standard deviation, median and interquartile range with 1st quartile and 3rd quartile.

Usage

mydescriptive(x)

Arguments

x

input numeric variable

Value

character vector

Examples

x=c(1:10)
  mydescriptive(x)
